# AI Image Generator Pro+ PDF Edition

A professional AI image generation tool with **real-time progressive previews**, **style presets**, **drag-and-drop thumbnail reordering**, and **PDF contact sheet export**.

## Features

- 4 variations per prompt
- Real-time progressive thumbnails
- Drag-and-drop thumbnail reordering
- Undo/Redo prompt history
- Style presets: Art, Photo, Cartoon, Anime
- Adjustable width, height, steps, and guidance
- Full-size preview on click
- Save individual or all images
- Export as PDF contact sheet with prompt text
- Automatic best-image selection using CLIP
